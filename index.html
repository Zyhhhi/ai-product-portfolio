Selected Work 01 重点项目 · AI 招聘人才库 MVP TalentFlow AI｜AI 面试人才库系统 MVP

面向 HR / HRBP 的 AI 面试人才库系统 MVP，展示对真实招聘业务场景的拆解、AI 工作流设计和 Vibe Coding 快速交付能力。

项目背景

根据 HR 提出的“面试人才库系统”需求，覆盖候选人录入、简历导入解析、面试流程跟进、AI 优劣势分析、招聘数据看板和内部试用版架构。

核心功能

设计候选人库表格管理、txt / docx / pdf 简历解析、简历信息自动提取、JD 匹配、AI 候选人评审工作台和招聘流程漏斗看板。

我的角色

负责 HR 场景拆解、产品流程、MVP 边界、AI 输出结构和验收标准设计，并使用 GPT + Codex 完成前端 MVP 交付与迭代。

技术栈

Vite / React / TypeScript / localStorage / Supabase / Cloudflare Worker / DeepSeek API / GitHub Pages。

项目亮点

不是通用聊天机器人，而是围绕“简历导入 → 候选人档案 → JD 匹配 → AI 分析 → 面试跟进 → 数据看板”的业务闭环设计。

后续迭代

可继续扩展数据库、权限、日志、简历文件存储、面试官协作和候选人隐私保护机制。
